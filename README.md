# theGargled Hosting

## 🌐 Overview
**theGargled Hosting** is a community‑funded platform offering **free web hosting, email, and reverse proxy services**.  
Our mission: *Hosting that stays free because we can.*  
Minimal ads, transparent rails, and community support keep the service alive.

---

## ✨ Features
- **Web Hosting**
  - Free SSL certificates
  - Subdomain included
  - One‑click install for WordPress and static sites

- **Email Hosting**
  - IMAP and SMTP access
  - Properly configured SPF, DKIM, and DMARC
  - Secure, standards‑compliant delivery

- **Reverse Proxy**
  - Routed and SSL‑terminated traffic
  - No manual certificate renewals
  - Seamless integration with hosted apps
